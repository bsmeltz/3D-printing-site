Ender 3 Pro

This is the 3D printer that I use. I have only made a few adjustments to it and it works really well. Here are a few of the adjustments I've made:

  • Added a cover to the screen and covered the lcd board

  • Upgraded the filament tube and nozzle

  • Added a filament guide, as well as bearings to a filament holder so the filament rolls smoothly

  • Added open-source software so I can control the stepper motors

  • Added guides and holders for the wires

  • And printed a custom fan blade when mine broke
